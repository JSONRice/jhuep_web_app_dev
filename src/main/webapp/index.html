Testing Simple Pages
  • hello.html Simple HTML page.
  • hello.jsp Simple JSP page.

Testing Simple Servlet
  • test1 The TestServlet, accessed via the URL given by the @WebServlet annotation in the Java source code.
  • test2 The TestServlet, accessed via the URL given in the web.xml file.

Form Data
  • three-params-form.html Form that sends data to the ThreeParams servlet.

JSP
  • sample.jsp JSP page that generates HTML: classic syntax. This syntax is preferred when generating full HTML pages.
  • sample.jspx JSP page that generates HTML: XML syntax. This syntax is usually more trouble than it is worth when generating full HTML pages.
  • some-xml-document.jspx JSP page that generates XML: XML syntax. Some developers prefer the XML syntax for JSP pages that build small chunks of XML in response to Ajax requests. Note that the JavaScript only cares that the output is legal HTML, but sometimes it is more convenient if the source code is also legal XML.
  • whats-new.jsp JSP page that uses jsp:include to build its output.

MVC
  • ran-form.html Form that sends data to the RandomNumberServlet.
